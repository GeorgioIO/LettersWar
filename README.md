# LettersWar

# ABOUT : 
LettersWar is a web trivia game where two team orange and green fight over a 5x5 board to capture cells , those cells have letters that represent an question that its answer start with that letter .

![Game Preview](assets/images/image.png)

# HOW TO WIN :
To win there are two cases normal win where either the orange capture a path from left to right or green from top to bottom.
in case of draw where no team captured a path , the team that captured the most cells wins.
